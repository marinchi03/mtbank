#mtbank
Требования к программе.

Есть вход для админа, осуществленна регистрация. Админ выбирает модуль "Клиенты" или "Депозиты"
При выборе модуля "Клиенты" возможно выполнение следующих действий:
- дополнить информацио о городе;
- удалить информацио о городе;
- дополнить информацио о гражданстве;
- удалить информацио о гражданстве;
- дополнить информацио об инвалидности;
- удалить информацио  об инвалидности;
- дополнить информацио о семейном положении;
- удалить информацио о семейном положении;
- добавить и удалить клиента с ранее добавленными данными;

При выблре модуля "Депозиты" возможно выполнение соледующих действий:
- дополнить информацию об основном счете;
- удалить информацио об основном счете;
- дополнить информацию о плане счетов;
- удалить информацио о плане счетов;
- дополнить информацию о виде опереций;
- удалить информацио о виде опереций;
- дополнить информацию о валюте;
- удалить информацио о валюте;
- получение информации о процентном счете;
- возможность закрыть день;
- заполнить информацию для договора на основе данных из модуля "Клиенты";
